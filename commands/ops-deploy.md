# ops-deploy

Handle deployment operations and infrastructure tasks

## Task
Manage deployment and operations including:
- Deployment pipeline configuration
- Environment setup (dev/staging/prod)
- Build optimization
- Docker/container configuration
- Cloud infrastructure (AWS/GCP/Azure/Vercel)
- Environment variables and secrets
- Monitoring and logging setup
- Rollback procedures
- Health checks and smoke tests

## Context
$ARGUMENTS

## Safety Checks
Before deployment:
1. Verify environment configuration
2. Review changes being deployed
3. Check dependency versions
4. Confirm backup/rollback plan
5. Validate monitoring is active

## Output
- Deployment plan
- Configuration files
- Documentation updates
- Post-deployment verification steps
